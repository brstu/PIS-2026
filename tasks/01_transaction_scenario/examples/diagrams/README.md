# PlantUML диаграммы последовательности

Эта папка содержит диаграммы последовательности для примера лабораторной работы #1.

## Файлы

### Успешный сценарий

- **`sequence-happy.puml`** — PlantUML исходник для happy path
- **`sequence-happy.png`** — Экспортированная диаграмма (PNG)

**Описание:** Полный успешный поток создания заявки на поиск и формирования группы с двумя транзакциями.

### Сценарий с ошибкой

- **`sequence-error-notification.puml`** — PlantUML исходник для error case
- **`sequence-error-notification.png`** — Экспортированная диаграмма (PNG)

**Описание:** Сценарий, где Notification Service (SMS) недоступен, но группа создаётся успешно, а уведомление ставится в очередь для retry.

## Как создать PNG из PlantUML?

### Вариант 1: VS Code плагин

1. Установите расширение **PlantUML** (jebbs.plantuml)
2. Откройте `.puml` файл
3. Нажмите `Alt+D` для preview
4. Нажмите `Ctrl+Shift+P` → "PlantUML: Export Current Diagram"
5. Выберите PNG формат

### Вариант 2: Онлайн редактор

1. Откройте https://www.plantuml.com/plantuml/uml/
2. Скопируйте содержимое `.puml` файла
3. Вставьте в редактор
4. Скачайте PNG через кнопку "PNG" над диаграммой

### Вариант 3: Командная строка (Java)

```bash
# Установите PlantUML
java -jar plantuml.jar sequence-happy.puml

# Создастся файл sequence-happy.png
```

### Вариант 4: Docker

```bash
docker run --rm -v ${PWD}:/data plantuml/plantuml sequence-happy.puml
```
